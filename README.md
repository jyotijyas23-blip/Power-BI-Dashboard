# Power-BI-Dashboard
**INSURANCE PERFORMANCE ANALYSIS** 
**This project analyzes insurance policy and claim data to understand**
**Claim status distribution (Rejected, Settled, Pending)**
Policy-wise premium performance
Coverage analysis by age group
Active vs Inactive policies
Gender-based filtering insights
Overall insurance business performance metrics
The interactive dashboard was built using Power BI to help stakeholders monitor claim trends, revenue distribution, and risk patterns.

**The goal of this project is to:**
Track total premium, coverage, and claim amount
Analyze claim settlement patterns
Identify high-performing policy types
Monitor active vs inactive policies
Understand demographic trends (Age & Gender).
